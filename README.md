# AI×ゴルフ 100切りチャレンジ 🏌️

3回の練習×AI分析で100切りを目指すWebアプリ

## 特徴
- 菅原大地プロ「最大効率スイング」理論ベース
- ハーフスイング習得 → ボディターン → 本番想定の3段階
- ショット記録 & AI分析フィードバック
- スマホ最適化UI

## デプロイ方法

### Vercel（推奨）
1. GitHubにリポジトリを作成してpush
2. Vercelでリポジトリをインポート
3. デプロイ完了

### ローカル開発
\`\`\`bash
npm install
npm run dev
\`\`\`

## 使い方
1. DAY1から順番に練習を実行
2. 各ショットの結果（Good/OK/Miss）を記録
3. セルフチェックを確認
4. AI分析でフィードバックを受ける
5. 3回の練習を完了して本番へ！
